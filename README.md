# Propojení strategií a rozpočtů

Webová prezentace zahraniční rešerše **Centra veřejných financí FSV UK** o propojování strategických dokumentů a státních rozpočtů prostřednictvím principů výkonového rozpočtování (performance budgeting).

---

## O projektu

Česko patří mezi posledních 5 zemí OECD, které nemají zavedeno výkonové rozpočtování. Tato rešerše mapuje, jak to dělají země, které to zvládly — a co z toho lze použít v českém prostředí.

Web je postaven jako informační a vzdělávací zdroj pro analytiky, úředníky a odbornou veřejnost se zájmem o reformu řízení veřejných financí.

---

## Obsah webu

| Sekce | Popis |
|---|---|
| **Úvod** | Zasazení do kontextu, klíčová čísla, přehled sekcí |
| **Kontext** | Co je výkonové rozpočtování, 3 typy přístupů, 7 principů OECD |
| **Zahraniční praxe** | Případové studie — Austrálie, Dánsko, Estonsko, Nizozemsko, Rakousko, Kanada |
| **Podmínky změny** | 5 procesních a institucionálních podmínek účinného propojení |
